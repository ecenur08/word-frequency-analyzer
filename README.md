# Word Frequency Analyzer

Bu proje Python ile yazılmış basit bir metin analiz uygulamasıdır.  
Bir metin dosyasını okuyarak kelime tekrar sayılarını hesaplar ve bazı istatistikler üretir.

## Projenin Yaptıkları

- Metindeki tüm kelimeleri küçük harfe çevirir
- Noktalama işaretlerini temizler
- Kelimelerin kaç kez geçtiğini hesaplar
- En çok tekrar eden ilk 3 kelimeyi bulur
- Uzunluğu 5 karakterden büyük kelimeler arasından en sık olanı belirler
- Sadece harflerden oluşan kelimeler arasından en çok geçen kelimeyi gösterir
- Toplam ve benzersiz kelime sayısını ekrana yazdırır

## Nasıl Çalıştırılır?

1. Proje klasörüne bir `.txt` dosyası eklenir.
2. Python dosyası çalıştırılır:

```bash
python word_frequency.py
